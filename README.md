OneBook是我自己想要做的一个命题,这是一个买卖书籍的平台
说明：
1、该网站用户为各大高校在校大学生和教师
2、用户可以在该网站买卖书籍
3、各大高校设有OneBook官方站点作为买卖书籍中转站

买书籍流程：
点击购买，付款----->等待书籍到站点----->去站点拿书籍，并确认收货
（ps:总用时不超过两天，最快能1小时之内完成交易）

卖书籍流程：
上传书籍信息到商城----->当书籍被购买后，把书籍拿到站点----->等待确认收货
（ps:总用时不超过两天，最快能1小时之内完成交易）
